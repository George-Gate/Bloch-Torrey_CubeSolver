# Torrey-Bloch_CubeSolver
Solving the Torrey-Bloch equation in Cubic domain.

Codes for research article "Systematic Errors from Inhomogeneous Alkali Polarization in NMR Gyroscopes and Comagnetometers".

Tha main package is in ```isotopeShift_Simu``` folder, ```figs``` folder contains the code for calculating and plotting the figures presented in the article.

# To Run the Codes
Add the following folder to MATLAB path:
 - isotopeShift_Simu
 - isotopeShift_Simu/chebfun-master
 - isotopeShift_Simu/specialFuns

See the following codes for demos:
 - isotopeShift_Simu/+Table0/CubeSolver2_demo.m
 - isotopeShift_Simu/+Table0/CubeSolver2_demo2_sweepCellTemperature_v2.m

The recommended MATLAB version is R2024a. Other versions newer than R2019b should work as well, though we have not performed comprehensive testing.
