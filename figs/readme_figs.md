`fig1.m` ~ `fig10.m` can be used to generate Figs.1 ~ 10 of the article. 

### Prerequisites
1. Add the folders inside `isotopeShift_Simu` to the MATLAB path, following the instructions in the project's README file.
2. To generate Fig. 8 or Fig. 9, first run the script `dataSet/total_drifts/prepareData.m` to obtain the data file `preparedData.mat`.
3. Run the desired script(s) (`fig1.m` ~ `fig10.m`) directly in their containing folder. The corresponding figure files will be created.

### Data
All data used to generate the figures are located in the `dataSet` folder. Each subfolder within `dataSet` contains:
- `.mat` files – the data used for plotting,
- `.m` scripts – the scripts that generated those `.mat` files.
